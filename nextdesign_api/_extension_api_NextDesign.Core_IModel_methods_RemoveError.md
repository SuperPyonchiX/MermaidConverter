IModel.RemoveError メソッド
=======================

名前空間: NextDesign.Core

説明​
-----------------------

このモデルに対して追加されているエラー情報を削除します。  
削除対象のエラー情報に null または、このモデルに含まれないエラー情報を指定した場合は、何も行われず正常終了します。

引数​
-----------------------

名前

型

説明

target

IError

削除するエラー情報  
nullが指定された場合は、何も行われません。

戻り値​
--------------------------

*   void
