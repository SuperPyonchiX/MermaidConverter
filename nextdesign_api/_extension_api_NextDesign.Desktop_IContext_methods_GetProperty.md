IContext.GetProperty メソッド
=========================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定された識別名のプロパティ値を取得します。  
該当する識別名が存在しない場合は null を返します。

引数​
-----------------------

名前

型

説明

name

string

識別名  
識別名には、空文字列を含むすべての文字列を指定できますが、nullは指定できません。

戻り値​
--------------------------

*   object

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

name に nullを指定した場合
