IRelationship.IsDerivation プロパティ
================================

名前空間: NextDesign.Core

説明​
-----------------------

導出関連か

注意）導出関連と関連端オブジェクト  
導出関連は、導出先から導出元方向への関連として定義されます。  
そのため、導出関連におけるSourceオブジェクトが導出先、Targetオブジェクトが導出元を表す点に注意してください。

    bool IsDerivation { get; }

*   bool
