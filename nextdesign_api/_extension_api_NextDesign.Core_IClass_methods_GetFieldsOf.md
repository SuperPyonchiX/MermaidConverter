IClass.GetFieldsOf メソッド
=======================

名前空間: NextDesign.Core

説明​
-----------------------

このクラスの指定された型のフィールドを取得します。  
このメソッドではプリミティブ型、および列挙型のフィールドを取得することはできません。  
プリミティブ型、および列挙型のフィールドをデータ型で取得したい場合は、GetFieldsByType() を使用してください。

引数​
-----------------------

名前

型

説明

type

IClass

フィールドの型（クラス型）

戻り値​
--------------------------

*   IFieldCollection
