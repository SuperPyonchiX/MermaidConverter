IEditorDef.GetElementsByTag メソッド
================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたタグが付与されたエディタ要素定義を取得します。

引数​
-----------------------

名前

型

説明

tag

string

タグ名

value

string

タグ値  
既定値は、null です。  
null または空文字が指定された場合は、エディタ要素に対するタグの有無のみで評価します。

戻り値​
--------------------------

*   IElementDefCollection
