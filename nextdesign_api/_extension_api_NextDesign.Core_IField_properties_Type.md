IField.Type プロパティ
=================

名前空間: NextDesign.Core

説明​
-----------------------

フィールドの型名

*   真偽値型 : "Boolean"
*   整数型 : "Integer"
*   実数型 : "Double"
*   文字列型 : "String"
*   リッチテキスト型 : "RichText"
*   列挙型 : 列挙の名前
*   クラス型 : クラスの名前

    string Type { get; }

型​
--------------------

*   string

注釈​
-----------------------

注意）  
列挙型、クラス型は、完全修飾名ではなく、型の名前のみを返します。
