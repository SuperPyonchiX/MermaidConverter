IProfile.GetProfileReferencePackageFor メソッド
===========================================

名前空間: NextDesign.Core

説明​
-----------------------

指定したプロファイル要素が所属するプロファイル参照パッケージを取得します。  
指定したプロファイル要素がプロファイル参照パッケージに所属していない場合、 プロファイルのルートパッケージを返します。

-----------------------

名前

型

説明

element

INamedElement

プロファイル要素
