ILifeline.SendMessages プロパティ
============================

名前空間: NextDesign.Core

説明​
-----------------------

このライフラインが送信元となるメッセージ一覧  
このライフライン上の全ての実行仕様の送信メッセージを結合した一覧を返します。  
コレクションの順序は以下のルールで並べ替えされます。

1.  このライフラインが保持する実行仕様の順序
2.  実行仕様の送信メッセージの順序

    IMessageCollection SendMessages { get; }

型​
--------------------

*   IMessageCollection
