IErrors.FindErrorByCategory メソッド
================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたカテゴリのエラー情報を検索します。  
検索結果のエラー情報にはサマリー情報も含みます。  
該当するエラー情報がない場合は空のコレクションを返します。

引数​
-----------------------

名前

型

説明

category

string

カテゴリ

戻り値​
--------------------------

*   IErrorCollection
