IRelationship.UnRelate メソッド
===========================

名前空間: NextDesign.Core

説明​
-----------------------

この関連による関連づけを解除して、この関連を削除します。  
既に削除済みの関連に対してこのメソッドが呼び出された場合は何も行われません。

なお、このメソッドは所有関連に対しては実行できません。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

不正操作

ExtensionInvalidOperationException

この関連が以下のいずれかの条件に当てはまる操作不可の関連である場合  
\- プロダクトラインのフィーチャ割り当て関連である  
\- System.Coreタグが付与されている関連である  
\- 所有関連
