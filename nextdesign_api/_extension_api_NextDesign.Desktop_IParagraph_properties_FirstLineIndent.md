IParagraph.FirstLineIndent プロパティ
================================

名前空間: NextDesign.Desktop.DocumentGenerator.Word

説明​
-----------------------

最初の行またはぶら下げインデントの値（ポイント単位）を取得または設定します。

正の値を使用して最初の行のインデントを設定し、負の値を使用してぶら下げインデントを設定します。

    double FirstLineIndent { get; set; }

型​
--------------------

*   double
