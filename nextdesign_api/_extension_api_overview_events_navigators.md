navigators イベントエリア
==================

説明​
-----------------------

ナビゲータの表示切替、選択変更を通知します。

イベント​
-----------------------------

名前

定義名

説明

[ナビゲータ表示イベント](_extension_api_overview_events_navigators_onShow.md)

onShow

ナビゲータを表示時に通知されるイベントです。

[ナビゲータ非表示イベント](_extension_api_overview_events_navigators_onHide.md)

onHide

ナビゲータを非表示時に通知されるイベントです。

[ナビゲータ内モデル選択イベント](_extension_api_overview_events_navigators_onSelectionChanged.md)

onSelectionChanged

ナビゲータ内でモデルを選択時に通知されるイベントです。
