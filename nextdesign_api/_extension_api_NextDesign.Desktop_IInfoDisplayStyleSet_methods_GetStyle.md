IInfoDisplayStyleSet.GetStyle メソッド
==================================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定された名前のスタイルの取得します。  
該当する名前のスタイルが存在しない場合は null を返します。

引数​
-----------------------

名前

型

説明

name

string

スタイル名  
  
null が指定された場合は、このメソッドの戻り値が null となります。

戻り値​
--------------------------

*   [IInfoDisplayStyle](_extension_api_NextDesign.Desktop_IInfoDisplayStyle.md)
