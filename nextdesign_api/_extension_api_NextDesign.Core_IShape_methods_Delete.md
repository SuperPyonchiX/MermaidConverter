IShape.Delete メソッド
==================

名前空間: NextDesign.Core

説明​
-----------------------

このシェイプを削除します。  
ただし、シーケンス図要素のシェイプは削除できません。

引数​
-----------------------

名前

型

説明

deleteModel

bool

このシェイプが対応するモデルも削除するか  
  
Trueの場合はこのシェイプとこのシェイプに対応するモデルを削除します。  
マッピング対象がフィールドのシェイプは、モデルを削除せずシェイプのみ削除することはできません。  
  
既定値はFalseです。

戻り値​
--------------------------

*   bool
