ICustomUI.OnBeforeDispose メソッド
==============================

名前空間: NextDesign.Desktop.CustomUI

説明​
-----------------------

独自拡張するユーザインタフェースを破棄する前の処理  
Next Designは、独自拡張するユーザインタフェースを破棄する前にこのメソッドを呼び出します。  
拡張側で破棄前に実行したい処理がある場合はここで実装します。

-----------------------

名前

型

説明

args

BeforeDisposeEventArgs
