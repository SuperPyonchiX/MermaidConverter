IMessage.Reply プロパティ
====================

名前空間: NextDesign.Core

説明​
-----------------------

このメッセージに対応する応答メッセージ  
対応する応答メッセージが存在しない場合は null を返します。

■ 対応する応答メッセージとは  
このメッセージの受信先の実行仕様が返す応答メッセージが該当します。

    IMessage Reply { get; }
