名前

[IUtility](_extension_api_NextDesign.Desktop_IUtility.md)

[IIndentableText](_extension_api_NextDesign.Desktop_IIndentableText.md)

テキストシーケンスのオブジェクトです。  
  
このインタフェースで定義されたメソッドは、全てメソッドが呼び出されたインスタンス自身を返します。  
これにより、メソッド呼び出しをチェインしてシーケンスを構成することができます。
