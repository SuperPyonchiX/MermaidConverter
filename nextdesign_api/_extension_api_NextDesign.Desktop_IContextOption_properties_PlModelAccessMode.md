IContextOption.PlModelAccessMode プロパティ
======================================

名前空間: NextDesign.Desktop

説明​
-----------------------

コンテキストが有効な期間における、モデル情報の取得・検索APIの振る舞い  
プロダクトラインがサポートされているプロジェクトにおいて、モデル情報を取得・検索する際に、プロダクトを評価するか否かを指定することができます。

    PlModelAccessMode PlModelAccessMode { get; set; }

型​
--------------------

*   PlModelAccessMode

注釈​
-----------------------

値域は次の通り。

*   "None" : 設定なし（既定）  
    　上位スコープのコンテキストオプションに従います。  
    　上位スコープのコンテキストオプションが存在しない場合は、AllProduct と同等の評価となります。
*   "AllProduct" : 150%モデル  
    　すべてのモデル情報の取得・検索が可能となります。
*   "CurrentProduct" : カレントプロダクト  
    　アプリケーションで指定されたカレントプロダクトで有効なモデル情報のみ取得・検索が可能となります。
*   "SpecifiedProduct" : 指定プロダクト  
    　別途指定するプロダクトで有効なモデル情報のみ取得・検索が可能となります。  
    　プロダクトの指定は、IContextOption.SpecifiedProductで実施します。
