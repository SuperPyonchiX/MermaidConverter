ISearch.CancelSearch メソッド
=========================

名前空間: NextDesign.Core

説明​
-----------------------

検索のキャンセルを要求します。  
キャンセルが要求された以降は、検索結果の登録を無視します。

引数​
-----------------------

名前

型

説明

flush

bool

それまでの検索結果を確定するか  
Trueが指定された場合、それまでの検索結果を確定します。  
  
既定値はFalseです。

戻り値​
--------------------------

*   void
