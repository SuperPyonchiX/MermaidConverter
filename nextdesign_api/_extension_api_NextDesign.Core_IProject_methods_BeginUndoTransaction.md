IProject.BeginUndoTransaction メソッド
==================================

名前空間: NextDesign.Core

説明​
-----------------------

編集を開始して、アンドゥトランザクションを生成します。  
アンドゥトランザクション内で実施された編集内容は、1回のアンドゥ/リドゥ操作の対象となります。

引数​
-----------------------

名前

型

説明

autoCommit

bool

トランザクションに対してコミット or ロールバックを指定せずに破棄した場合に、自動的にコミットを行うかを指定します。  
falseを指定した場合はロールバックが行われます。  
既定値は true です。
