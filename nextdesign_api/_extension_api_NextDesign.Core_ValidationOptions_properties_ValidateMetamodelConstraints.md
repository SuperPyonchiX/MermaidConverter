ValidationOptions.ValidateMetamodelConstraints プロパティ
====================================================

名前空間: NextDesign.Core

説明​
-----------------------

メタモデルの整合検証(多重度検証、パス制約検証)を行うかを取得、または設定します。  
既定値は true です。

    bool ValidateMetamodelConstraints { get; set; }

型​
--------------------

*   bool
