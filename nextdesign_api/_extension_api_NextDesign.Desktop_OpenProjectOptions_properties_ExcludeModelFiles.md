OpenProjectOptions.ExcludeModelFiles プロパティ
==========================================

名前空間: NextDesign.Desktop

説明​
-----------------------

モデルファイルを読み込まずにプロジェクトを開くかを指定できます。

True が指定された場合、モデルファイルを読み込まずにプロジェクトを開きます。  
既定値は False です。

    bool ExcludeModelFiles { get; set; }

型​
--------------------

*   bool
