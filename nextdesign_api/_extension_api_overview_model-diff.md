モデル差分・比較
========

説明​
-----------------------

モデルの比較操作や差分情報にアクセスするAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[IDiff](_extension_api_NextDesign.Core_IDiff.md)

任意のプロジェクト間でのモデル情報の比較操作を提供します。

[IModelComparison](_extension_api_NextDesign.Core_IModelComparison.md)

比較処理単位情報へのアクセスオブジェクトです。

[IMatch](_extension_api_NextDesign.Core_IMatch.md)

比較情報へのアクセスオブジェクトです。

[IDifference](_extension_api_NextDesign.Core_IDifference.md)

差分情報へのアクセスオブジェクトです。
