ICustomUI.Descriptor プロパティ
==========================

名前空間: NextDesign.Desktop.CustomUI

説明​
-----------------------

記述子を取得、または設定します。  
登録時に指定したDescriptorが生成時に設定されます。  
未設定の場合、カスタムUIは表示されません。

    ICustomDescriptor Descriptor { get; set; }

型​
--------------------

*   [ICustomDescriptor](_extension_api_NextDesign.Desktop_ICustomDescriptor.md)
