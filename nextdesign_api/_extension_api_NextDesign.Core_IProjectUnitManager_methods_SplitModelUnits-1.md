IProjectUnitManager.SplitModelUnits(IEnumerable<IModel>) メソッド
=============================================================

名前空間: NextDesign.Core

説明​
-----------------------

指定された全てのモデルをユニットファイルに分割します。  
分割されたユニットファイルのファイル名は、モデル名を用いて重複しないファイル名を自動的に決定します。

引数​
-----------------------

名前

型

説明

models

IEnumerable<IModel>

モデルの列挙

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

不正なパラメータが指定された場合  
\- 指定されたモデルにルート要素が含まれる  
\- 指定されたモデルに既にユニット分割済みの要素が含まれる

不正操作

ExtensionInvalidOperationException

無効な操作要求の場合  
\- 最新のモデルファイルが読み込まれていない  
\- プロジェクトの編集権限がない

注釈​
-----------------------

指定されたモデルの列挙を分割する際、ユニットファイルは分割元モデルと同じ保存形式で分割します。  
他の保存形式で分割したい場合、現在のプロジェクトの保存形式を変更してください。  
また、ファイル単位差分比較中に呼び出した場合、比較有無のずれが生じることがあるため、再比較を実行する必要があります。
