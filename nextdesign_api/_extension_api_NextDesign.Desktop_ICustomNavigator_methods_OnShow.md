ICustomNavigator.OnShow メソッド
============================

名前空間: NextDesign.Desktop.CustomUI

説明​
-----------------------

このナビゲータを表示する際の処理  
Next Designは、独自拡張するナビゲータを表示する際にこのメソッドを呼び出します。  
拡張側で表示時に実行したい処理がある場合はここで実装します。
