onModelEdited イベント
==================

説明​
-----------------------

モデルを編集時に通知されるイベントです。

APIイベント引数型​
-----------------------------------------------

*   [ModelEditedEventParams](_extension_api_NextDesign.Desktop_ModelEditedEventParams.md)

注釈​
-----------------------

このイベントは、現在アプリケーションで編集対象として開かれているプロジェクト（"IWorkspace.CurrentProject"）で変更された内容のみ通知します。これは、他の「モデル」イベントエリアのイベントとは異なる動作です。他の「モデル」イベントエリアのイベントは、アプリケーションで編集対象として開かれているプロジェクトに限らず、すべてのプロジェクトで変更された内容をイベントで通知します。
