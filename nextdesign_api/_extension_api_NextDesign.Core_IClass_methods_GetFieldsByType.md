IClass.GetFieldsByType メソッド
===========================

名前空間: NextDesign.Core

説明​
-----------------------

このクラスの指定された型名のフィールドを取得します。  
クラス型フィールドは、クラスの完全修飾名を指定することで取得できます。  
列挙型フィールドは、列挙の完全修飾名を指定することで取得できます。

引数​
-----------------------

名前

型

説明

typeName

string

フィールドの型名  
\- "Boolean" : 真偽値型  
\- "Integer" : 整数型  
\- "Double" : 実数型  
\- "String" : 文字列型  
\- "RichText" : リッチテキスト型  
\- その他の任意文字列 : クラス、または列挙の完全修飾名として扱います。

戻り値​
--------------------------

*   IFieldCollection
