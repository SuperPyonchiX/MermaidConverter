ICommonUI.ShowSaveFileDialog メソッド
=================================

名前空間: NextDesign.Desktop

説明​
-----------------------

ファイルの保存ダイアログを表示して、ダイアログで選択したファイルのパスを取得します。  
ダイアログがキャンセルされた場合はnullを返します。

引数​
-----------------------

名前

型

説明

title

string

ダイアログのタイトル（省略した場合はアプリケーション名を使用します）

filter

string

保存先を選択する際のフィルタ（省略した場合は全てのファイルを対象とします）

initialPath

string

保存するファイル名の初期値（省略した場合は初期値なしとなります）

戻り値​
--------------------------

*   string
