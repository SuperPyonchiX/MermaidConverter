IPort.ChangeOrientation メソッド
============================

名前空間: NextDesign.Core

説明​
-----------------------

このポートの配置方向を指定された方向に変更します。

引数​
-----------------------

名前

型

説明

orientation

string

配置方向  
\- "Top" : 上  
\- "Bottom" : 下  
\- "Left" : 左  
\- "Right" : 右

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

orientation に配置方向として許可されない位置が指定された場合
