IProductLineModel.ApplyProductBy メソッド
=====================================

名前空間: NextDesign.Core

説明​
-----------------------

指定された名前のプロダクトを適用して、カレントプロダクトに設定します。

引数​
-----------------------

名前

型

説明

productName

string

プロダクト名。  
適用プロダクトなし（150%モデル）とする場合は、"$Master"を指定します。

--------------------------

*   void
