IMessageShape.SelfloopBendsX プロパティ
==================================

名前空間: NextDesign.Core

説明​
-----------------------

自己接続メッセージにおけるベンド位置のX座標  
自己接続メッセージでない場合は、常に0が返ります。  
自己接続メッセージは、送信元と受信先のライフラインが同一となるメッセージです。  
ベンド位置は、該当メッセージにおける最初の折れ線のポイントとなります。

    double SelfloopBendsX { get; }

型​
--------------------

*   double
