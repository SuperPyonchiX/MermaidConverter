ITreeGridNode.GetCellDisplayValues メソッド
=======================================

名前空間: NextDesign.Core

説明​
-----------------------

このノードのすべてのセル表示文字列を取得します。  
値が存在しないセルは空の文字列を返します。

取得できる文字列は、GetCellValueString() を参照してください。

戻り値​
--------------------------

*   IList<string>

注釈​
-----------------------

C#のエクステンション限定のAPIです。
