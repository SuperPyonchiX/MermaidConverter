OpenProjectOptions.IsSetCurrent プロパティ
=====================================

名前空間: NextDesign.Desktop

説明​
-----------------------

ロードしたプロジェクトをカレントプロジェクトに設定するかを指定できます。

False が指定された場合、ロードしたプロジェクトはカレントプロジェクトとして設定されません。  
既定値は True です。

    bool IsSetCurrent  { get; set; }

型​
--------------------

*   bool
