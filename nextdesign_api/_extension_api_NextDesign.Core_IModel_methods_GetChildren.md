IModel.GetChildren メソッド
=======================

名前空間: NextDesign.Core

説明​
-----------------------

このインスタンスの直接の所有関係にある所有先インスタンスを取得します。  
このメソッドは、IContextOption.PlModelAccessMode を評価します。

このインスタンスと直接の所有関係にある所有先インスタンスのコレクションを返します。  
該当するインスタンスが存在しない場合は空のコレクションを返します。

戻り値​
--------------------------

*   IModelCollection
