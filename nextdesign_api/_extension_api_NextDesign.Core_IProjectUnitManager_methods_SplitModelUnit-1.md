IProjectUnitManager.SplitModelUnit(IModel,string) メソッド
======================================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたモデルを指定された名前のユニットファイルに分割します。

引数​
-----------------------

名前

型

説明

model

IModel

モデル

unitName

string

ユニットファイル名

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

不正なパラメータが指定された場合  
\- 指定されたモデルがルート要素である  
\- 指定されたモデルが既にユニット分割済みである  
\- Modelsフォルダ直下にあるユニットファイル名と重複する  
\- 現在のオペレーションシステムで使用できないユニットファイル名が指定された

不正操作

ExtensionInvalidOperationException

無効な操作要求の場合  
\- 最新のモデルファイルが読み込まれていない  
\- プロジェクトの編集権限がない

注釈​
-----------------------

指定されたモデルを指定された名前のユニットファイルに分割する際、ユニットファイルは分割元モデルと同じ保存形式で分割します。  
他の保存形式で分割したい場合、現在のプロジェクトの保存形式を変更してください。  
また、ファイル単位差分比較中に呼び出した場合、比較有無のずれが生じることがあるため、再比較を実行する必要があります。
