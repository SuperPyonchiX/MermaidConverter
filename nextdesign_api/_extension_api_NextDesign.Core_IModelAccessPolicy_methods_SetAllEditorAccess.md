IModelAccessPolicy.SetAllEditorAccess メソッド
==========================================

名前空間: NextDesign.Core

説明​
-----------------------

全てのエディタの詳細情報を取得可能とするか設定します。  
取得可能に設定した場合、エディタの内部構造を解析するため、エディタAPIの応答に解析時間が加算されます。

引数​
-----------------------

名前

型

説明

enabled

bool

全てのエディタの詳細情報を取得可能とするか  
取得可能とする場合は true を指定します。

戻り値​
--------------------------

*   void
