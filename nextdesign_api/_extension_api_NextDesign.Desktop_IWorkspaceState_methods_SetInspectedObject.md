IWorkspaceState.SetInspectedObject メソッド
=======================================

名前空間: NextDesign.Desktop

説明​
-----------------------

インスペクト対象の要素を設定します。

引数​
-------------------------

名前

型

説明

target

object

対象要素  
  
対象要素を設定時に表示するインスペクタタブは下記の通りです。  
・エディタページ表示中  
　プロジェクト直下のモデル: プロパティ、関連タブ  
　モデル：プロパティ、関連、メタモデルタブ  
　プロダクトラインモデル：プロパティタブ  
　メタモデル：メタモデルタブ  
　パッケージ：パッケージタブ  
　列挙型：列挙型タブ  
　フィールド：フィールドタブ  
　ダイアグラムエディタ：ダイアグラム定義、メタモデルタブ  
　ノードシェイプ：プロパティ、関連、シェイプ定義、メタモデルタブ  
　コネクタシェイプ：プロパティ、関連、メタモデルタブ  
　シーケンス図シェイプ：プロパティ、関連、シェイプ定義、メタモデルタブ  
　フォーム要素(モデル自体を表現するもの)：プロパティ、関連、フォーム要素、メタモデルタブ  
　フォーム要素(フィールドを表現するもの)：フォーム要素、フィールドタブ  
　ダイアグラム定義：ダイアグラム定義タブ  
　シェイプ定義：シェイプ定義タブ  
　シーケンス図要素定義：シェイプ定義タブ  
　フォーム定義：フォーム定義タブ  
　フォーム要素定義：フォーム要素タブ  
　その他：(表示タブなし)  
・トレースページ表示中  
　モデル：関連タブ  
　その他：(表示タブなし)

戻り値​
--------------------------

*   void
