IOutput インタフェース
===============

名前空間: NextDesign.Desktop

説明​
-----------------------

出力サービスへのアクセス手段を提供します。

所属エリア​
--------------------------------

名前

説明

[検索・エラー・出力](_extension_api_overview_errors.md)

エラー情報や検索結果、出力ウィンドウにアクセスするAPI群です。

メソッド​
-----------------------------

名前

説明

[Clear](_extension_api_NextDesign.Desktop_IOutput_methods_Clear.md)

指定されたカテゴリの出力をクリアします。

[ClearAll](_extension_api_NextDesign.Desktop_IOutput_methods_ClearAll.md)

すべての出力をクリアします。

[WriteFormatLine](_extension_api_NextDesign.Desktop_IOutput_methods_WriteFormatLine.md)

出力の指定されたカテゴリに指定のフォーマットで文字列を追加します。  
なお、フォーマット文字列の置換についてはC#のString.Formatに準拠します。

[WriteLine](_extension_api_NextDesign.Desktop_IOutput_methods_WriteLine.md)

出力の指定されたカテゴリに文字列を追加します。
