ModelReferableResult インタフェース
============================

名前空間: NextDesign.Core.EditingCapabilities

説明​
-----------------------

モデル参照時の選択対象の結果を表すオブジェクトです。

Next Designは、この結果オブジェクトを用いて選択対象を構築します。
