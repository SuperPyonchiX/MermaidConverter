IModelComparison.GetMatch メソッド
==============================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたモデルの比較情報を取得します。

引数​
-----------------------

名前

型

説明

model

IModel

モデル  
ここでは、比較元、比較先のいずれのモデルを指定しても同一のオブジェクトを取得することができます。  
例えば、比較元にのみ存在するモデルの比較情報を取得する場合は、比較元側のモデルを指定することで比較情報を取得することができます。
