==============================

-----------------------

"RichTextBox" : リッチテキストボックス  
"CheckBox" : チェックボックス  
"ComboBox" : コンボボックス  
"List" : リスト  
"Grid" : グリッド  
"ModelReferenceControl" : モデル参照コントロール  
"EntityControl" : エンティティコントロール  
"GroupControl" : グループコントロール  
"GridRow" : グリッド行

    string ControlType { get; }
