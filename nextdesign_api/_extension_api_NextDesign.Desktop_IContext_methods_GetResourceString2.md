IContext.GetResourceString2 メソッド
================================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定されたリソースキーの文字列を取得します。  
このときリソース文字列で定義された置換子"{0},{1}"を、それぞれ引数で指定された param1, param2 の文字列表現に置き換えて返します。  
なお、リソースパラメータの置換についてはC#のString.Formatに準拠します。  
また、リソース文字列特定方法、およびリソースキーの指定の詳細は、GetResourceString()を参照してください。

引数​
-----------------------

名前

型

説明

key

string

リソースキー

param1

object

リソースパラメータの置換オブジェクト  
リソース文字列内の置換子"{0}"に置き換えるオブジェクト  
null を指定した場合は、文字列 "null" に置き換わります。

param2

object

リソースパラメータの置換オブジェクト  
リソース文字列内の置換子"{1}"に置き換えるオブジェクト  
null を指定した場合は、文字列 "null" に置き換わります。

戻り値​
--------------------------

*   string
