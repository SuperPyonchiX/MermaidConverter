ICommonUI.ShowOpenFileDialog メソッド
=================================

名前空間: NextDesign.Desktop

説明​
-----------------------

ファイルを開くダイアログを表示して、ダイアログで選択したファイルのパスを取得します。  
ダイアログがキャンセルされた場合はnullを返します。

引数​
-----------------------

名前

型

説明

title

string

ダイアログのタイトル（省略した場合はアプリケーション名を使用します）

filter

string

ファイルを選択する際のフィルタ（省略した場合は全てのファイルを対象とします）

戻り値​
--------------------------

*   string
