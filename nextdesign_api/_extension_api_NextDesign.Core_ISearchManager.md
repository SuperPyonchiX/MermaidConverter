--------------------------------

説明

メソッド​
-----------------------------

名前

説明

[ClearResults](_extension_api_NextDesign.Core_ISearchManager_methods_ClearResults.md)

全ての検索結果情報をクリアします。

[Create](_extension_api_NextDesign.Core_ISearchManager_methods_Create.md)

検索サービスオブジェクトを生成します。

[FindResultByModel](_extension_api_NextDesign.Core_ISearchManager_methods_FindResultByModel.md)

与えられたモデルを対象とした検索結果情報を検索します。

[RemoveResult](_extension_api_NextDesign.Core_ISearchManager_methods_RemoveResult.md)

指定した検索結果を削除します。  
既に削除済みの検索結果を指定した場合は何も行いません。

[RemoveResults](_extension_api_NextDesign.Core_ISearchManager_methods_RemoveResults.md)

指定した検索結果をすべて削除します。  
既に削除済みの検索結果が含まれる場合は、その検索結果を無視します。
