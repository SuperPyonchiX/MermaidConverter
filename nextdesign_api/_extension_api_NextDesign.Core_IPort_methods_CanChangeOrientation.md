IPort.CanChangeOrientation メソッド
===============================

名前空間: NextDesign.Core

説明​
-----------------------

このポートを指定方向に配置できるか調べます。

引数​
-----------------------

名前

型

説明

orientation

string

配置方向  
\- "Top" : 上  
\- "Bottom" : 下  
\- "Left" : 左  
\- "Right" : 右  
配置方向に未知の値が指定された場合は、false を返します。

戻り値​
---------------------------

*   bool
