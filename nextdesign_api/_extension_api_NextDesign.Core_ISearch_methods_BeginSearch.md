ISearch.BeginSearch メソッド
========================

名前空間: NextDesign.Core

説明​
-----------------------

検索開始を要求します。

引数​
-----------------------

名前

型

説明

name

string

検索名  
  
以降、この検索サービスでは、ここで指定した検索名が設定されます。

type

string

検索種別  
  
以降、この検索サービスで追加した検索結果には、ここで設定した検索種別が設定されます。

append

bool

追加検索とするか  
Trueを指定した場合は、前回の検索結果を維持したまま次の検索を開始します。  
  
既定値はFalseです。

戻り値​
--------------------------

*   void
