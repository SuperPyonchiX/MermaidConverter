IFileUtility インタフェース
====================

名前空間: NextDesign.Desktop

説明​
-----------------------

ファイル操作を提供します。

所属エリア​
--------------------------------

名前

説明

[ユーティリティ](_extension_api_overview_utility.md)

汎用API群です。

メソッド​
-----------------------------

名前

説明

[CreateFolder](_extension_api_NextDesign.Desktop_IFileUtility_methods_CreateFolder.md)

指定されたパスのフォルダを生成します。  
戻り値は生成したフォルダのパスです。

[CreateTextFile](_extension_api_NextDesign.Desktop_IFileUtility_methods_CreateTextFile.md)

指定されたフォルダに、指定された名前、および拡張子のテキストファイルを生成します。  
戻り値は生成したファイルのパスです。

[GetFolder](_extension_api_NextDesign.Desktop_IFileUtility_methods_GetFolder.md)

指定されたファイルの格納フォルダのパスを取得します。

[WriteAllText](_extension_api_NextDesign.Desktop_IFileUtility_methods_WriteAllText.md)

指定されたファイルに与えられたテキストを書き出します。
