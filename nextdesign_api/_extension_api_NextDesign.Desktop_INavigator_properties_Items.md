INavigator.Items プロパティ
======================

名前空間: NextDesign.Desktop

説明​
-----------------------

このナビゲータの全ての管理要素

現在のバージョンでは、以下のナビゲータのみこのプロパティをサポートします。

*   モデルナビゲータ
*   プロダクトラインナビゲータ
*   プロジェクトナビゲータ

モデルナビゲータとプロダクトラインナビゲータはIModel型のコレクションを返します。  
プロジェクトナビゲータはIModelUnit型のコレクションを返します。

    ICollection Items { get; }

型​
--------------------

*   ICollection

注釈​
-----------------------

プロジェクトナビゲータでツリーのトップレベルに表示されている \[プロジェクト\]、\[プロファイル\]、\[モデル\] はこのプロパティに含まれません。

部分ロードノード以下のアイテムを取得でき、プロジェクトノード以下のアイテム群の後に部分ロードノード以下のアイテムが含まれます。

*   例）以下の場合、Items=\[projectA, modelA, modelB, modelB1, modelX, modelY, modelY1\]となります。
    *   projectA
        *   modelA
        *   modelB
            *   modelB1
    *   部分ロードノード
        *   modelX
        *   modelY
            *   modelY1
