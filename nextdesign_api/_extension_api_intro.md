Next Design の API について
======================

説明​
-----------------------

Next Design のエクステンション開発に利用できる API を機能エリア別に説明します。

API仕様のプレビュー公開

マークの付いている API は先行公開しているものです。 なお、現時点でこれらのAPIは品質保証しておりませんので、ご利用される場合はユーザー様の責任でご利用ください。 また、これらのAPI仕様は予告なく変更する場合がありますのでご了承ください。
