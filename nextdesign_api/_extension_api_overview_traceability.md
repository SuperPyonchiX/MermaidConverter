トレース
====

説明​
-----------------------

トーレス情報にアクセスするAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[ITracePage](_extension_api_NextDesign.Desktop_ITracePage.md)

トレースUIへのアクセス手段を提供します。

[ITraceView](_extension_api_NextDesign.Desktop_ITraceView.md)

トレースビュー情報へのアクセス手段を提供します。

[ITraceCoverageExcludedModel](_extension_api_NextDesign.Desktop_ITraceCoverageExcludedModel.md)

トレース除外対象のモデル情報  
トレース除外対象と設定されたモデルは、トレース網羅率の計算対象から除外されます。

[ITraceLane](_extension_api_NextDesign.Desktop_ITraceLane.md)

トレース対象のルートオブジェクトを管理するレーン（系列）情報へのアクセス手段を提供します。

[ITraceNode](_extension_api_NextDesign.Desktop_ITraceNode.md)

トレース対象のモデルに対応するノード情報へのアクセス手段を提供します。
