IModel.SetProductApplyCondition メソッド
====================================

名前空間: NextDesign.Core

説明​
-----------------------

このモデルのプロダクト適用条件式を設定します。  
なお、プロダクト適用条件式内において、このモデルに未割り当てのフィーチャ名を使用していた場合は、自動的にフィーチャが割り当てられます。  
また、条件式で、このモデルに割り当て済みのフィーチャ名が使用されなかった場合は、自動的にフィーチャとの割り当てが解除されます。  
条件式に空の文字列を指定した場合は、全てのフィーチャ割り当てが解除されます。

名前

型

expression

string

条件式

例外​
-----------------------

名前

例外クラス

説明

不正なプロダクト条件式

ExtensionFormulaException

次のような不正なプロダクト適用条件式が指定された場合  
\- 条件式のシンタックスエラー  
\- 存在しないフィーチャ名を含む  
\- 一意に識別できないフィーチャ名を含む

注釈​
-----------------------

プロダクト適用条件式の書式は、GetProductApplyCondition() を参照してください。
