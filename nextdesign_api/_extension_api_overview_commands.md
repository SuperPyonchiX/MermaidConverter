コマンド
====

説明​
-----------------------

コマンドハンドラで受け取ったコマンドにアクセスするAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[ICommandManager](_extension_api_NextDesign.Desktop_ICommandManager.md)

Extensionで追加登録するコマンド、およびアプリケーションが提供するシステムコマンドを管理します。

[ICommand](_extension_api_NextDesign.Desktop_ICommand.md)

コマンド定義情報を提供します。

[ICommandContext](_extension_api_NextDesign.Desktop_ICommandContext.md)

コマンドの実行コンテキストを提供します。

[ICommandParams](_extension_api_NextDesign.Desktop_ICommandParams.md)

コマンドのパラメータを提供します。
