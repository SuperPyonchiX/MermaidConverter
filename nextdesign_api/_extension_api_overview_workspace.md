ワークスペース・プロジェクト
==============

説明​
-----------------------

アプリケーションの作業領域やアプリケーションで開いているプロジェクトにアクセスするAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[IWorkspace](_extension_api_NextDesign.Desktop_IWorkspace.md)

アプリケーションの作業領域情報へのアクセス手段を提供します。

[IProject](_extension_api_NextDesign.Core_IProject.md)

プロジェクト情報へのアクセス手段を提供します。

[ProjectExtensions](_extension_api_NextDesign.Core_ProjectExtensions.md)

IProject の拡張クラスです。

[IModelUnit](_extension_api_NextDesign.Core_IModelUnit.md)

モデルユニット情報のアクセスオブジェクトです。

[IProjectUnitManager](_extension_api_NextDesign.Core_IProjectUnitManager.md)

プロジェクトの物理ファイル構成管理オブジェクトです。

[IWorkspaceState](_extension_api_NextDesign.Desktop_IWorkspaceState.md)

ワークスペースの状態管理オブジェクトです。

[BeforeProjectCloseEventArgs](_extension_api_NextDesign.Desktop_BeforeProjectCloseEventArgs.md)

ワークスペース上で、プロジェクトが閉じられようとしていることを通知するイベントパラメータ

[DisplayModeChangedEventArgs](_extension_api_NextDesign.Desktop_DisplayModeChangedEventArgs.md)

表示モードの状態変更イベントパラメータ

[IDisplayMode](_extension_api_NextDesign.Desktop_IDisplayMode.md)

ワークスペースにおける表示に関する各種設定情報

[ProjectOpenedEventArgs](_extension_api_NextDesign.Desktop_ProjectOpenedEventArgs.md)

ワークスペース上で、プロジェクトが開かれたことを通知するイベントパラメータ

[WorkspaceStateChangedEventArgs](_extension_api_NextDesign.Desktop_WorkspaceStateChangedEventArgs.md)

ワークスペース状態の変更イベントパラメータ
