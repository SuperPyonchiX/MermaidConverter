IModel.UnRelateByFieldTag メソッド
==============================

名前空間: NextDesign.Core

説明​
-----------------------

このインスタンスの指定したタグが付与された参照フィールドで指定したモデルとの関連づけを解除します。  
指定したモデルとの関連が存在しなかった場合は、何も行われず正常終了します。  
また、以下の条件に該当するフィールドへの関連づけも解除されず正常終了します。  
・プロダクトラインサポート向けフィールドでの関連付け  
・System.Coreタグが付与されているフィールドでの関連付け

引数​
-----------------------

名前

型

説明

opposite

IModel

関連づけを解除するモデル  
null は指定できません。

fieldTag

string

フィールドのタグ名  
null、または空文字列が指定された場合は、何も行われません。

value

string

タグ値の文字列表現  
既定値は、null です。  
null または空文字が指定された場合は、フィールドに対するタグの有無のみで評価します。
