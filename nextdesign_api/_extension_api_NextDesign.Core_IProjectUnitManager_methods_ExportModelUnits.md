IProjectUnitManager.ExportModelUnits メソッド
=========================================

名前空間: NextDesign.Core

説明​
-----------------------

指定された全てのモデルユニットを指定したフォルダパスにエクスポートします。  
エクスポート先のファイル名は元のユニットファイル名と同一となります。

引数​
-----------------------

名前

型

説明

units

IEnumerable<IModelUnit>

ユニットの列挙

exportFolderPath

string

エクスポート先フォルダパス

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

不正操作

ExtensionInvalidOperationException

無効な操作要求の場合  
\- 指定されたユニットがプロジェクト管理下でない  
\- プロジェクトが一度も保存されていない  
\- 指定されたユニットがロードされていない
