カスタムUI
======

説明​
-----------------------

カスタムUIにアクセスするAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[ICustomUIRegistry](_extension_api_NextDesign.Desktop_ICustomUIRegistry.md)

独自拡張のユーザインタフェースのレジストリです。

[ICustomUI](_extension_api_NextDesign.Desktop_ICustomUI.md)

エクステンションにより独自拡張するユーザインタフェースに共通のプロパティ、メソッドを定義します。

[CustomEditorConfigs](_extension_api_NextDesign.Desktop_CustomEditorConfigs.md)

カスタムエディタの特性を静的に決定するコンフィグレーションです。

[ICustomEditorView](_extension_api_NextDesign.Desktop_ICustomEditorView.md)

エクステンションにより独自拡張するカスタムエディタの実装用インタフェースです。  
Next Design では、このインタフェースを介してカスタムエディタとの情報の交換を行います。

[BeforeDisposeEventArgs](_extension_api_NextDesign.Desktop_BeforeDisposeEventArgs.md)

独自拡張したユーザインタフェースが破棄されようとしている際に通知するイベントパラメータ

[CustomUIEventArgs](_extension_api_NextDesign.Desktop_CustomUIEventArgs.md)

独自拡張したユーザインタフェースへ通知するイベントパラメータの基底

[InitializedEventArgs](_extension_api_NextDesign.Desktop_InitializedEventArgs.md)

独自拡張したユーザインタフェースを初期化する際に通知するイベントパラメータ

[OnHideEventArgs](_extension_api_NextDesign.Desktop_OnHideEventArgs.md)

独自拡張したユーザインタフェースを非表示にする際に通知するイベントパラメータ

[OnShowEventArgs](_extension_api_NextDesign.Desktop_OnShowEventArgs.md)

独自拡張したユーザインタフェースを表示にする際に通知するイベントパラメータ

[CustomInspectorConfigs](_extension_api_NextDesign.Desktop_CustomInspectorConfigs.md)

カスタムインスペクタの特性を静的に決定するコンフィグレーションです。

[ICustomInspector](_extension_api_NextDesign.Desktop_ICustomInspector.md)

エクステンションにより独自拡張するカスタムインスペクタの実装用インタフェースです。  
Next Design では、このインタフェースを介してカスタムインスペクタとの情報の交換を行います。

[CustomNavigatorConfigs](_extension_api_NextDesign.Desktop_CustomNavigatorConfigs.md)

[ICustomNavigator](_extension_api_NextDesign.Desktop_ICustomNavigator.md)

Next Design では、このインタフェースを介してカスタムナビゲータとの情報の交換を行います。
