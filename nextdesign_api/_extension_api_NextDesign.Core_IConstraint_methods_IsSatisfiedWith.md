IConstraint.IsSatisfiedWith メソッド
================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたモデルを基点に、与えられたオブジェクトがこの制約に適合するか調べます。  
制約を満たす場合は、true を返します。

なお、制約評価の基点となるモデルに、この制約と無関係のモデルが指定された場合は、常に true を返すように評価されます。

引数​
-----------------------

名前

型

説明

model

IModel

制約評価の基点となるモデル

target

object

制約を評価する対象オブジェクト

戻り値​
--------------------------

*   bool
