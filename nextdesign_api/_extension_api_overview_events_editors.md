editors イベントエリア
===============

説明​
-----------------------

エディタの表示切替、選択変更を通知します。

イベント​
-----------------------------

名前

定義名

説明

[エディタ表示イベント](_extension_api_overview_events_editors_onShow.md)

onShow

エディタを表示時に通知されるイベントです。

[エディタ非表示イベント](_extension_api_overview_events_editors_onHide.md)

onHide

エディタを非表示時に通知されるイベントです。

[エディタ内モデル選択イベント](_extension_api_overview_events_editors_onSelectionChanged.md)

onSelectionChanged

エディタ内でモデルを選択時に通知されるイベントです。
