ISequenceDiagram.GetShapeById メソッド
==================================

名前空間: NextDesign.Core

説明​
-----------------------

指定された識別子のシェイプを取得します。  
該当する識別子のシェイプが存在しない場合は null を返します。

引数​
-----------------------

名前

型

説明

shapeIdentifier

string

シェイプの識別子

戻り値​
--------------------------

*   [ISequenceShape](_extension_api_NextDesign.Core_ISequenceShape.md)
