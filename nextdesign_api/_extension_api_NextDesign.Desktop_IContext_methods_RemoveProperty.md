IContext.RemoveProperty メソッド
============================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定された識別子名のプロパティ値を削除します。  
該当する識別名のプロパティ値が存在しない場合は、何も行われません。

引数​
-----------------------

名前

型

説明

name

string

識別名  
識別名には、空文字列を含むすべての文字列を指定できますが、nullは指定できません。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

name に nullを指定した場合
