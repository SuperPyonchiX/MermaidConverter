IProduct インタフェース
================

名前空間: NextDesign.Core

説明​
-----------------------

プロダクト情報対するアクセスオブジェクトです。

所属エリア​
--------------------------------

名前

説明

[プロダクトライン](_extension_api_overview_productline.md)

プロダクトラインモデルにアクセスするAPI群です。

継承元​
--------------------------

名前

説明

[IModel](_extension_api_NextDesign.Core_IModel.md)

NextDesignの設計モデル情報へのアクセス手段を提供します。

プロパティ​
--------------------------------

名前

説明

[SelectedFeatures](_extension_api_NextDesign.Core_IProduct_properties_SelectedFeatures.md)

このプロダクトで選択されているフィーチャ一覧  
選択要素のコレクションの順序は不定です。

メソッド​
-----------------------------

名前

説明

[DeselectFeature](_extension_api_NextDesign.Core_IProduct_methods_DeselectFeature.md)

指定されたフィーチャを未選択にします。  
選択状態を設定できた場合はTrueを返します。

[DeselectFeatureByName](_extension_api_NextDesign.Core_IProduct_methods_DeselectFeatureByName.md)

指定された名前のフィーチャを未選択にします。  
選択状態を設定できた場合はTrueを返します。

[DeselectFeatures](_extension_api_NextDesign.Core_IProduct_methods_DeselectFeatures.md)

指定されたすべてのフィーチャを未選択にします。  
選択状態を設定できた場合はTrueを返します。

[DeselectFeaturesByName](_extension_api_NextDesign.Core_IProduct_methods_DeselectFeaturesByName.md)

指定された名前のすべてのフィーチャを未選択にします。  
選択状態を設定できた場合はTrueを返します。

[SelectFeature](_extension_api_NextDesign.Core_IProduct_methods_SelectFeature.md)

指定されたフィーチャを選択します。  
選択状態を設定できた場合はTrueを返します。

[SelectFeatureByName](_extension_api_NextDesign.Core_IProduct_methods_SelectFeatureByName.md)

指定された名前のフィーチャを選択します。  
選択状態を設定できた場合はTrueを返します。

[SelectFeatures](_extension_api_NextDesign.Core_IProduct_methods_SelectFeatures.md)

指定されたすべてのフィーチャを選択します。  
選択状態を設定できた場合はTrueを返します。

[SelectFeaturesByName](_extension_api_NextDesign.Core_IProduct_methods_SelectFeaturesByName.md)

指定された名前のすべてのフィーチャを選択します。  
選択状態を設定できた場合はTrueを返します。
