IProductLineModel.ExportAppliedProject メソッド
===========================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたプロダクトを適用したプロジェクトを指定されたパスにエクスポートします。  
エクスポートしたプロジェクトは以下の状態となります。

*   プロファイルはエクスポート元と同一
*   指定されたプロダクトで有効なモデル要素、エディタのみが存在する
*   プロダクトラインモデル（フィーチャモデル、コンフィグレーションモデル）はなし
*   ユニット分割はなし

引数​
-----------------------

名前

型

説明

product

IProduct

プロダクト。  
nullは指定できません。

exportProjectPath

string

プロジェクトの出力先のパス。  
絶対パスで指定してください。

戻り値​
--------------------------

*   void

-----------------------

名前

例外クラス

説明
