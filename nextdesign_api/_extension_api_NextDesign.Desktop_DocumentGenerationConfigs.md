DocumentGenerationConfigs インタフェース
=================================

名前空間: NextDesign.Desktop.DocumentGenerator

説明​
-----------------------

ドキュメント生成のコンフィギュレーションです。

所属エリア​
--------------------------------

名前

説明

[ドキュメント生成](_extension_api_overview_documents.md)

ドキュメント生成にアクセスするAPI群です。

プロパティ​
--------------------------------

名前

説明

[OnAfterContentsGeneration](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnAfterContentsGeneration.md)

ドキュメント生成モデル生成後のコールバック関数を取得します。

[OnAfterContentsWrite](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnAfterContentsWrite.md)

コンテンツ毎の書き込み後のコールバック関数を取得します。

[OnBeforeContentWrite](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnBeforeContentWrite.md)

コンテンツ毎の書き込み前のコールバック関数を取得します。

[OnDocumentWriteFinish](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnDocumentWriteFinish.md)

ドキュメントの書き込み終了後のコールバック関数を取得します。

[OnDocumentWriteStart](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnDocumentWriteStart.md)

ドキュメントの書き込み開始前のコールバック関数を取得します。

[OnGenerationFinish](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnGenerationFinish.md)

ドキュメント生成の終了後のコールバック関数を取得します。

[OnGenerationStart](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_OnGenerationStart.md)

ドキュメント生成の開始前のコールバック関数を取得します。

[Options](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_Options.md)

ドキュメントの生成オプションを取得します。

[Priority](_extension_api_NextDesign.Desktop_DocumentGenerationConfigs_properties_Priority.md)

プライオリティを取得または設定します。
