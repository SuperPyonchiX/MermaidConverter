IEditorView.SelectedModels プロパティ
================================

名前空間: NextDesign.Desktop

説明​
-----------------------

エディタで選択されているモデル  
選択要素のコレクションの順序は不定です。  
選択されているモデルがない場合は、空のコレクションを返します。

    IModelCollection SelectedModels { get; }

型​
--------------------

*   IModelCollection

注釈​
-----------------------

このプロパティでは、差分エディタの選択モデルは取得できません。  
差分エディタ側の選択状態を取得する場合は、DiffEditorを使用してください。
