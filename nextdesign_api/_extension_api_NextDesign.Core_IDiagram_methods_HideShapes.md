IDiagram.HideShapes メソッド
========================

名前空間: NextDesign.Core

説明​
-----------------------

指定された全てのシェイプを非表示にします。  
マッピング対象がクラスのシェイプが含まれている場合、そのシェイプは削除します。

引数​
-----------------------

名前

型

説明

shapes

IEnumerable<IShape>

非表示にするシェイプの列挙。  
null は指定できません。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

shapes に null　を指定した場合
