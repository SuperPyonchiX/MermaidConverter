ITracePage.SelectCell メソッド
==========================

名前空間: NextDesign.Desktop

説明​
-----------------------

表示しているビューの種別が Matrix の場合、指定されたセルを選択します。  
行、列で指定したノードが、それぞれのレーンに存在しない場合は何も行われません。  
また、レーン上のノードは選択されません。

なお、表示しているビューの種類が Tree の場合は何も行いません。

引数​
-----------------------

名前

型

説明

row

ITraceNode

選択するセルの行のノード

column

ITraceNode

選択するセルの列のノード

append

bool

現在のセル選択状態を維持して追加選択するか  
true を指定した場合は、現在のセル選択状態を維持して追加選択します。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

row または column に null を指定した場合
