IModel.GetRelatingFieldsOf メソッド
===============================

名前空間: NextDesign.Core

説明​
-----------------------

このインスタンスのクラスが持つ全てのフィールドのうち、指定したモデルをフィールド値として格納するものを取得します。  
このメソッドで取得できるフィールドは、所有/参照に関係なく取得します。

引数​
-----------------------

名前

型

説明

model

IModel

対象モデル

戻り値​
--------------------------

*   IFieldCollection
