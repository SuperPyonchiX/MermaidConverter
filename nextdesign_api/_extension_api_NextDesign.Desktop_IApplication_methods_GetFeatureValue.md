IApplication.GetFeatureValue メソッド
=================================

名前空間: NextDesign.Desktop

説明​
-----------------------

現在のエディションにおける、指定したフィーチャの指定したキー（属性）値を取得します。  
指定したフィーチャの有効/無効に関係なく値を取得できます。

なお、指定したフィーチャが見つからない場合、値はnullを返します。  
また、指定したフィーチャキー名が見つからない場合、値はnullを返します。

引数​
-----------------------

名前

型

説明

featureName

string

フィーチャ識別名

key

string

フィーチャキー（属性）名

戻り値​
--------------------------

*   object
