INavigator.Select メソッド
======================

名前空間: NextDesign.Desktop

説明​
-----------------------

このナビゲータで指定された要素を選択します。

現在のバージョンでは、以下のナビゲータのみこのメソッドをサポートします。

*   モデルナビゲータ
*   プロダクトラインナビゲータ

引数​
-----------------------

名前

型

説明

item

object

選択する要素  
  
現在のバージョンでは、IModelのインスタンスのみ指定することができます。

append

bool

追加選択とするか  
Trueを指定すると、現在の選択に追加して指定された要素を選択します。  
既定値は False です。  
  
ナビゲータが複数要素選択に対応していない場合は、この指定は無視されます。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

INavigator.Itemsに存在しない要素が指定された場合
