IViewItemContent インタフェース
========================

名前空間: NextDesign.Desktop.DocumentGenerator

説明​
-----------------------

ドキュメントのビューの項目を表すコンテンツです。

所属エリア​
--------------------------------

名前

説明

[ドキュメント生成](_extension_api_overview_documents.md)

ドキュメント生成にアクセスするAPI群です。

継承元​
--------------------------

名前

説明

[IDocumentContent](_extension_api_NextDesign.Desktop_IDocumentContent.md)

ドキュメントのコンテンツです。

プロパティ​
--------------------------------

名前

説明

[Control](_extension_api_NextDesign.Desktop_IViewItemContent_properties_Control.md)

ビューのUI部品を取得または設定します。

[Field](_extension_api_NextDesign.Desktop_IViewItemContent_properties_Field.md)

フィールドを取得します。  
  
フィールドのビューの場合のみ値が設定されます。

[Items](_extension_api_NextDesign.Desktop_IViewItemContent_properties_Items.md)

ビューの子要素のリストを取得します。

[Model](_extension_api_NextDesign.Desktop_IViewItemContent_properties_Model.md)

モデルを取得します。  
  
モデルのビューの場合のみ値が設定されます。

[OutlineLevel](_extension_api_NextDesign.Desktop_IViewItemContent_properties_OutlineLevel.md)

現在のアウトラインレベルを取得します。

メソッド​
-----------------------------

名前

説明

[ResetOutlineLevel](_extension_api_NextDesign.Desktop_IViewItemContent_methods_ResetOutlineLevel.md)

アウトラインレベルを末端まで降りなおします。

[SetModel](_extension_api_NextDesign.Desktop_IViewItemContent_methods_SetModel.md)

モデルを設定します。
