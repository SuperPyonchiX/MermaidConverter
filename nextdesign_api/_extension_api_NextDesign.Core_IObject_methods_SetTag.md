IObject.SetTag メソッド
===================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたタグ名／値でタグを設定します。

*   タグ名に一致するタグが存在しない場合はタグを追加します
*   タグ名に一致するタグが存在する場合は値を変更します

引数​
-----------------------

名前

型

説明

tag

string

タグ名

value

string

タグ値  
未指定の場合はタグ有無のみで評価するタグを設定します。

戻り値​
--------------------------

*   void

注釈​
-----------------------

このAPIでは編集権限をチェックしないため、読み取り専用のモデルに対してもタグを編集できます。  
読み取り専用のモデルに対して設定したタグは保存されないため、プロジェクトを閉じるとタグの編集結果は失われます。
