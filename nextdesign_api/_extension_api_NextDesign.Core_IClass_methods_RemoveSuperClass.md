IClass.RemoveSuperClass メソッド
============================

名前空間: NextDesign.Core

説明​
-----------------------

指定したクラスをこのクラスのスーパークラスから削除します。  
指定したクラスがこのクラスのスーパークラスに含まれない場合は何も行われません。  
クラスの継承関係を削除すると継承先クラスのモデルも削除します。

型

説明
