IProjectUnitManager.ImportModelUnits(IEnumerable<string>) メソッド
==============================================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたファイルをモデルユニットとしてインポートします。  
ユニットファイルが、Modelsフォルダ直下に存在しない場合は、該当のファイルをModelsフォルダ直下にコピーしてインポートします。  
ユニットファイルが、Modelsフォルダ直下に存在する場合は、そのファイルをインポートします。

引数​
-----------------------

名前

型

説明

unitFilePaths

IEnumerable<string>

ユニットファイルの絶対パスの列挙

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

プロジェクトの保存形式と指定されたユニットファイルパスの拡張子が対応しない場合

不正操作

ExtensionInvalidOperationException

無効な操作要求の場合  
\- 最新のモデルファイルが読み込まれていない  
\- プロジェクトの編集権限がない  
\- プロジェクトが一度も保存されていない
