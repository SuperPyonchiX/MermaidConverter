ITreeGridNode.GetCellValueStringAt メソッド
=======================================

名前空間: NextDesign.Core

説明​
-----------------------

このノードの指定されたインデックスのセルの値を文字列形式で取得します。  
セルの値が存在しない場合は空の文字列を返します。

取得できる文字列は、GetCellValueString() を参照してください。

引数​
-----------------------

名前

型

説明

index

int

列インデックス（0以上、列数未満）

戻り値​
--------------------------

*   string

例外​
-----------------------

名前

例外クラス

説明

範囲不正

ExtensionArgumentException

指定したインデックス範囲が不正な場合
