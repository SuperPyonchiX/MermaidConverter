IModel.GetDerivedModels(IRelationshipClass) メソッド
================================================

名前空間: NextDesign.Core

説明​
-----------------------

指定した関連クラスにより、このインスタンスから導出したインスタンスを取得します。  
このメソッドは、IContextOption.PlModelAccessMode を評価します。

-----------------------

名前

型

説明

targetClass

IRelationshipClass

導出関連クラス

戻り値​
--------------------------
