IFinder.ShowFinder メソッド
=======================

名前空間: NextDesign.Desktop

説明​
-----------------------

ファインダを表示して選択されたモデルを取得します。

-----------------------

名前

型

説明

model

IModel

関連付け対象モデル

fieldName

string

関連付けるフィールド

string
