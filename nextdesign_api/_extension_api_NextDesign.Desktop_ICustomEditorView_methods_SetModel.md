ICustomEditorView.SetModel メソッド
===============================

名前空間: NextDesign.Desktop.CustomUI

説明​
-----------------------

このエディタの表示対象のモデルを設定します。

引数​
-----------------------

名前

型

説明

model

ICustomEditor

モデル  
エディタの永続化情報を保持するカスタムエディタモデルが与えられます。  
独自定義のエディタの表示に必要な情報は、このモデルを介してアクセスすることができます。

戻り値​
--------------------------

*   void
