名前

プロパティ​
--------------------------------

名前

説明

[AutoFitTables](_extension_api_NextDesign.Desktop_WordOptions_properties_AutoFitTables.md)

テーブルの列幅を自動調整する方法を取得または設定します。  
  
nullの場合は、自動調整しません。

[OrderByViewDefinition](_extension_api_NextDesign.Desktop_WordOptions_properties_OrderByViewDefinition.md)

ビューの出力順をビュー定義順にするかを取得または設定します。  
  
ビュー定義順としない場合、ダイアグラム・シーケンス図を先に出力します。
