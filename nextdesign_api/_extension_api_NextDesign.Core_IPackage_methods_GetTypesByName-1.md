IPackage.GetTypesByName<T>(string,bool) メソッド
============================================

名前空間: NextDesign.Core

説明​
-----------------------

このパッケージ配下から指定した名前のいずれかに一致する型を全て取得します。  
型名には、カンマ区切りで複数の型名を指定できます。  
複数の型名を指定した場合は、そのいずれかに一致していれば対象となります。

recursive に`true`を指定した場合は、深さ優先探索でサブパッケージも探索の対象とします。

-----------------------------------

入力型

制約

T

IType

-----------------------

名前

型

説明

typeNames

string

型名  
カンマ区切りで複数の型名を指定できます。  
  
注意）型名に完全修飾名を指定した場合はヒットしません。

recursive

bool

サブパッケージも探索の対象とするか  
既定値は false です。
