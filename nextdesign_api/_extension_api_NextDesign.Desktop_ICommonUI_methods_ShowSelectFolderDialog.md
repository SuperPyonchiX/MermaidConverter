ICommonUI.ShowSelectFolderDialog メソッド
=====================================

名前空間: NextDesign.Desktop

説明​
-----------------------

フォルダを開くダイアログを表示して、ダイアログで選択したフォルダのパスを取得します。  
ダイアログがキャンセルされた場合はnullを返します。

引数​
-----------------------

名前

型

説明

title

string

ダイアログのタイトル（省略した場合はアプリケーション名を使用します）

initialPath

string

初期選択するフォルダパス（省略した場合は、前回選択したフォルダを初期選択します）

戻り値​
--------------------------

*   string
