ITreeGrid.GetSelectedNodes メソッド
===============================

名前空間: NextDesign.Core

説明​
-----------------------

エディタで選択されているツリーノードを取得します。  
選択要素のコレクションの順序は不定です。

引数​
-----------------------

名前

型

説明

excludeSelectedChildren

bool

このパラメータに真を指定すると、ツリーノードの親要素が選択されている場合は子要素の選択状態に関わらず除外されます

戻り値​
--------------------------

*   ITreeGridNodeCollection
