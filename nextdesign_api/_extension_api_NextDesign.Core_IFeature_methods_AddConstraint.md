IFeature.AddConstraint メソッド
===========================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたフィーチャとの間に指定した種類の制約関係を追加します。

引数​
-----------------------

名前

型

説明

target

IFeature

制約先フィーチャ

kind

string

制約種類  
\- 排他関係 : "Conflicts"  
端点のフィーチャが同時に有効とすることができない関係を表します。  
\- 相互依存関係 : "Implicit"  
端点のフィーチャが必ず同時に有効となる関係を表します。  
\- 依存関係: "Requires"  
依存元が有効な場合、必ず依存先が有効となる関係を表します。逆方向は不定です。

戻り値​
--------------------------

*   void
