INodeShapeStyle.SetImage メソッド
=============================

名前空間: NextDesign.Core

説明​
-----------------------

シェイプに画像を設定します。  
シェイプ定義で画像の貼り付けが許可されていない場合は実行しても画像変更されず無視されます。

引数​
-----------------------

名前

型

説明

data

byte\[\]

画像データのバイト配列

format

string

画像データのフォーマット。"png", "bmp", "jpg", "gif"、大文字小文字は区別しません。

戻り値​
--------------------------

*   void
