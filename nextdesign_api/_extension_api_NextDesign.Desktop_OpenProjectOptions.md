OpenProjectOptions インタフェース
==========================

名前空間: NextDesign.Desktop

説明​
-----------------------

オープンプロジェクトオプション一覧。

プロパティ​
--------------------------------

名前

説明

[AddToRecentFiles](_extension_api_NextDesign.Desktop_OpenProjectOptions_properties_AddToRecentFiles.md)

開いたプロジェクトを最近使ったファイルに追加するかを指定できます。  
  
False が指定された場合、開いたプロジェクトは最近使ったファイルに追加されません。  
既定値は True です。

[ExcludeModelFiles](_extension_api_NextDesign.Desktop_OpenProjectOptions_properties_ExcludeModelFiles.md)

モデルファイルを読み込まずにプロジェクトを開くかを指定できます。  
  
True が指定された場合、モデルファイルを読み込まずにプロジェクトを開きます。  
既定値は False です。

[IsSetCurrent](_extension_api_NextDesign.Desktop_OpenProjectOptions_properties_IsSetCurrent_.md)

ロードしたプロジェクトをカレントプロジェクトに設定するかを指定できます。  
  
False が指定された場合、ロードしたプロジェクトはカレントプロジェクトとして設定されません。  
既定値は True です。

[ReadOnly](_extension_api_NextDesign.Desktop_OpenProjectOptions_properties_ReadOnly.md)

読み取り専用でプロジェクトを開くかを指定できます。  
  
True が指定された場合、開いたプロジェクトは編集できません。  
既定値は False です。
