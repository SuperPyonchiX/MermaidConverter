ITraceView インタフェース
==================

名前空間: NextDesign.Desktop

説明​
-----------------------

トレースビュー情報へのアクセス手段を提供します。

所属エリア​
--------------------------------

名前

説明

[トレース](_extension_api_overview_traceability.md)

トーレス情報にアクセスするAPI群です。

プロパティ​
--------------------------------

名前

説明

[Id](_extension_api_NextDesign.Desktop_ITraceView_properties_Id.md)

識別子

[Name](_extension_api_NextDesign.Desktop_ITraceView_properties_Name.md)

ビュー名  
表示対象のビュー設定の名前を返します。

[TraceLanes](_extension_api_NextDesign.Desktop_ITraceView_properties_TraceLanes.md)

トレースビューに表示する系列（個別のツリー情報）のコレクション  
ビューでのツリー表示順序でコレクションに格納されます。  
  
なお、ビューの種類が Matrix の場合は、要素の1つ目が行、2つ目が列に対応し、それ以外の系列は使用されません。
