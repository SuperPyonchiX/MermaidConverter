IAsyncValidationContext.Options プロパティ
=====================================

名前空間: NextDesign.Core

説明​
-----------------------

検証オプションを取得します。  
この検証オプションはコンテキスト生成時に指定された検証オプションのスナップショットです。  
ここで取得したオプションに対して設定を変更しても検証処理には反映されません。

    ValidationOptions Options { get; }

型​
--------------------

*   [ValidationOptions](_extension_api_NextDesign.Core_ValidationOptions.md)
