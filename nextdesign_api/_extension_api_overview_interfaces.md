ユーザインタフェース
==========

説明​
-----------------------

エディタやナビゲータなどのUIにアクセスするAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[IWorkspaceWindow](_extension_api_NextDesign.Desktop_IWorkspaceWindow.md)

アプリケーションのウィンドウへのアクセス手段を提供します。

[IEditorPage](_extension_api_NextDesign.Desktop_IEditorPage.md)

エディタUIへのアクセス手段を提供します。

[IUIElement](_extension_api_NextDesign.Desktop_IUIElement.md)

UI要素へのアクセス手段を提供します。

[IEditorView](_extension_api_NextDesign.Desktop_IEditorView.md)

メイン・サブエディタUIの共通インタフェースです。

[INavigator](_extension_api_NextDesign.Desktop_INavigator.md)

ナビゲータへのアクセス手段を提供します。

[IInfoView](_extension_api_NextDesign.Desktop_IInfoView.md)

情報ウィンドウの表示ページへのアクセス手段を提供します。

[ICommonUI](_extension_api_NextDesign.Desktop_ICommonUI.md)

基本的なダイアログなどのUIへのアクセス手段を提供します。

[IFinder](_extension_api_NextDesign.Desktop_IFinder.md)

ファインダUIへのアクセス手段を提供します。
