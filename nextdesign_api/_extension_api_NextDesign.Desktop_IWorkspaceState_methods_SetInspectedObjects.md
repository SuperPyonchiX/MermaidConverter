IWorkspaceState.SetInspectedObjects メソッド
========================================

名前空間: NextDesign.Desktop

説明​
-----------------------

インスペクト対象の要素（複数）を設定します。

引数​
------------------------

名前

型

説明

targets

IEnumerable<object>

対象要素の列挙  
  
対象要素を設定時に表示するインスペクタタブはSetInspectedObject()を参照してください。  
モデル以外の要素を複数指定して設定すると、列挙の最後の要素のインスペクタタブを表示します。

戻り値​
--------------------------

*   void
