IListFormat インタフェース
===================

名前空間: NextDesign.Desktop.DocumentGenerator.Word

説明​
-----------------------

段落に適用されるリストの書式を制御します。

所属エリア​
--------------------------------

名前

説明

[ドキュメント生成](_extension_api_overview_documents.md)

ドキュメント生成にアクセスするAPI群です。

メソッド​
-----------------------------

名前

説明

[ApplyBulletDefault](_extension_api_NextDesign.Desktop_IListFormat_methods_ApplyBulletDefault.md)

新しいデフォルトの箇条書きリストを開始し、それを段落に適用します。

[ApplyNumberDefault](_extension_api_NextDesign.Desktop_IListFormat_methods_ApplyNumberDefault.md)

新しいデフォルトの番号付きリストを開始し、それを段落に適用します。

[ListIndent](_extension_api_NextDesign.Desktop_IListFormat_methods_ListIndent.md)

現在の段落のリストレベルを1レベル上げます。

[ListOutdent](_extension_api_NextDesign.Desktop_IListFormat_methods_ListOutdent.md)

現在の段落のリストレベルを1レベル下げます。

[RemoveNumbers](_extension_api_NextDesign.Desktop_IListFormat_methods_RemoveNumbers.md)

現在の段落から数字または箇条書きを削除し、リストレベルをゼロに設定します。
