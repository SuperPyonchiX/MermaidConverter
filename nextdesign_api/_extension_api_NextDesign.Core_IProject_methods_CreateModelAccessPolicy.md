IProject.CreateModelAccessPolicy メソッド
=====================================

名前空間: NextDesign.Core

説明​
-----------------------

モデルアクセスのポリシーを作成します。  
ポリシーが有効な範囲において行われたモデルへのアクセスには、ポリシーの設定が適用されます。
