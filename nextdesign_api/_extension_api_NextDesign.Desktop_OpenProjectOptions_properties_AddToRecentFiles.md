OpenProjectOptions.AddToRecentFiles プロパティ
=========================================

名前空間: NextDesign.Desktop

説明​
-----------------------

開いたプロジェクトを最近使ったファイルに追加するかを指定できます。

False が指定された場合、開いたプロジェクトは最近使ったファイルに追加されません。  
既定値は True です。

    bool AddToRecentFiles { get; set; }

型​
--------------------

*   bool
