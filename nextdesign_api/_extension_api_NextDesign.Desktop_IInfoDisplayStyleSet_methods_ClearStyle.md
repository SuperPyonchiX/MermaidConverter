IInfoDisplayStyleSet.ClearStyle メソッド
====================================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定された名前のスタイルをクリアします。  
該当するスタイルが存在しない場合は何も行われません。

引数​
-----------------------

名前

型

説明

name

string

スタイル名  
  
null が指定された場合は、何も行われません。

戻り値​
--------------------------

*   void
