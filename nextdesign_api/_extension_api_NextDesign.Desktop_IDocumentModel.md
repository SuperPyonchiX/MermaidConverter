IDocumentModel インタフェース
======================

説明​
-----------------------

ドキュメントの文章構造を表すコンテンツです。

所属エリア​
--------------------------------

名前

説明

[ドキュメント生成](_extension_api_overview_documents.md)

ドキュメント生成にアクセスするAPI群です。

プロパティ​
--------------------------------

名前

説明

[Pages](_extension_api_NextDesign.Desktop_IDocumentModel_properties_Pages.md)

ページのリストを取得します。  
  
対象範囲が「選択要素のみ出力する」以外の場合に設定されます。

[Title](_extension_api_NextDesign.Desktop_IDocumentModel_properties_Title.md)

ドキュメントタイトルを取得または設定します。

[Views](_extension_api_NextDesign.Desktop_IDocumentModel_properties_Views.md)

ビューのリストを取得します。  
  
対象範囲が「選択要素のみ出力する」の場合に設定されます。
