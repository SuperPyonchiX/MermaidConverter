IContext.SetProperty メソッド
=========================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定された識別名のプロパティ値を設定します。  
既に存在する識別名が指定された場合は、プロパティ値を与えられた値で更新します。

引数​
-----------------------

名前

型

説明

name

string

識別名  
識別名には、空文字列を含むすべての文字列を指定できますが、nullは指定できません。

value

object

プロパティ値  
null を含むすべてのオブジェクトを指定できます。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

name に nullを指定した場合
