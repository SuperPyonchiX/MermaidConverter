IProjectUnitManager.ExportModelUnit メソッド
========================================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたモデルユニットを指定したファイルパスでエクスポートします。

引数​
-----------------------

名前

型

説明

unit

IModelUnit

ユニット

exportFilePath

string

エクスポート先ファイルパス

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

不正なパラメータが指定された場合  
\- 指定されたエクスポート先のファイルパスが、対象ユニットと同一  
\- プロジェクトの保存形式と指定されたエクスポート先のファイルパスの拡張子が対応しない場合

不正操作

ExtensionInvalidOperationException

無効な操作要求の場合  
\- 指定されたユニットがプロジェクト管理下でない  
\- プロジェクトが一度も保存されていない  
\- 指定されたユニットがロードされていない
