ICellFormat インタフェース
===================

名前空間: NextDesign.Desktop.DocumentGenerator.Word

説明​
-----------------------

セルのフォーマットを制御します。

所属エリア​
--------------------------------

名前

説明

[ドキュメント生成](_extension_api_overview_documents.md)

ドキュメント生成にアクセスするAPI群です。

プロパティ​
--------------------------------

名前

説明

[BackgroundPatternColor](_extension_api_NextDesign.Desktop_ICellFormat_properties_BackgroundPatternColor.md)

セルの背景に適用される色を取得または設定します。

[BottomPadding](_extension_api_NextDesign.Desktop_ICellFormat_properties_BottomPadding.md)

セルの内容の下に追加するスペースの量（ポイント単位）を取得または設定します。

[FitText](_extension_api_NextDesign.Desktop_ICellFormat_properties_FitText.md)

セルのテキストをセル内に収めるようにするかを取得または設定します。

[HorizontalMerge](_extension_api_NextDesign.Desktop_ICellFormat_properties_HorizontalMerge.md)

セルを行内の他のセルと水平方向に結合する方法を取得または設定します。

[LeftPadding](_extension_api_NextDesign.Desktop_ICellFormat_properties_LeftPadding.md)

セルの内容の左側に追加するスペースの量（ポイント単位）を取得または設定します。

[RightPadding](_extension_api_NextDesign.Desktop_ICellFormat_properties_RightPadding.md)

セルの内容の右側に追加するスペースの量（ポイント単位）を取得または設定します。

[TopPadding](_extension_api_NextDesign.Desktop_ICellFormat_properties_TopPadding.md)

セルの内容の上に追加するスペースの量（ポイント単位）を取得または設定します。

[VerticalMerge](_extension_api_NextDesign.Desktop_ICellFormat_properties_VerticalMerge.md)

セルを他のセルと垂直方向に結合する方法を取得または設定します。

[Width](_extension_api_NextDesign.Desktop_ICellFormat_properties_Width.md)

セルの幅（ポイント単位）を取得または設定します。

[WrapText](_extension_api_NextDesign.Desktop_ICellFormat_properties_WrapText.md)

セルのテキストを折り返すかを取得または設定します。

メソッド​
-----------------------------

名前

説明

[ClearFormatting](_extension_api_NextDesign.Desktop_ICellFormat_methods_ClearFormatting.md)

デフォルトのセルフォーマットにリセットします。  
  
セルの幅は変更されません。
