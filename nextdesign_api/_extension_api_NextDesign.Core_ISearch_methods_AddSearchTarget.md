ISearch.AddSearchTarget メソッド
============================

名前空間: NextDesign.Core

説明​
------------------------

検索対象を追加します。  
検索対象の情報は検索結果ウィンドウやエディタ、ナビゲータ等で表示されます。

引数​
-----------------------

名前

型

説明

model

IModel

検索対象とするモデル  
  
nullを指定することはできません。

戻り値​
--------------------------

*   void

例外​
-----------------------

名前

例外クラス

説明

引数不正

ExtensionArgumentException

model に null が指定された場合

注釈​
-----------------------

現バージョンでは、このメソッドによる検索対象の追加は未対応です。  
検索対象情報は使われません。
