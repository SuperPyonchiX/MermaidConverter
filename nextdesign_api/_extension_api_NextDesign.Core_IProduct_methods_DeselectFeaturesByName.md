IProduct.DeselectFeaturesByName メソッド
====================================

名前空間: NextDesign.Core

説明​
-----------------------

指定された名前のすべてのフィーチャを未選択にします。  
選択状態を設定できた場合はTrueを返します。

引数​
-----------------------

名前

型

説明

featureNames

string

フィーチャ名（カンマ区切りで複数指定可）

戻り値​
--------------------------

*   bool
