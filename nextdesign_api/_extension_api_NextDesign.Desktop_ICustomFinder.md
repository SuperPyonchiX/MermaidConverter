ICustomFinder インタフェース
=====================

名前空間: NextDesign.Desktop.CustomUI

説明​
-----------------------

エクステンションにより独自拡張するカスタムファインダの実装用インタフェースです。  
Next Design では、このインタフェースを介してカスタムファインダとの情報の交換を行います。
