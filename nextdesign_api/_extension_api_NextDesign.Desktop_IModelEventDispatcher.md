IModelEventDispatcher インタフェース
=============================

名前空間: NextDesign.Desktop

説明​
-----------------------

モデルのプロパティ変更通知を受信するメカニズムを提供します。

所属エリア​
---------------------------------

名前

説明

[モデル](_extension_api_overview_model.md)

モデルにアクセスするAPI群です。

メソッド​
-----------------------------

名前

説明

[ClearEventHandler](_extension_api_NextDesign.Desktop_IModelEventDispatcher_methods_ClearEventHandler.md)

指定されたオブザーバで登録された全てのイベントハンドラの登録を解除します

指定されたオブザーバにおいて、指定されたモデルのプロパティ変更通知を受信するイベントハンドラを登録します。
