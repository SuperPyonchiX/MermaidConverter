IApplication.Restart メソッド
=========================

名前空間: NextDesign.Desktop

説明​
-----------------------

アプリケーションの再起動を要求します。  
再起動要求時にプロジェクトを開いていた場合は、再起動後に同じプロジェクトを開きます。  
なお、再起動に失敗した場合は false を返します。

戻り値​
--------------------------

*   bool

注釈​
-----------------------

プロジェクトを開いた状態で呼び出された場合、該当プロジェクトが保存されていなくても警告なく破棄します。  
また、呼び出しが成功した場合、この呼び出しを行ったプロセスが終了するため、以降の処理は実行されません。
