IExtensions.Reload メソッド
=======================

名前空間: NextDesign.Desktop

説明​
-----------------------

指定された識別子のエクステンションをリロードします。  
reloadManifest に true を指定するとマニフェストを再ロードして、リボン等のUIも再構築します。  
reloadManifest に false を指定した場合は、コード（ソースコード）、言語リソースなどのメモリ保持データをクリアして再ロードします。この時、リボン等のUIは更新されません。

引数​
-----------------------

名前

型

説明

extensionIdentifier

string

エクステンション識別子

reloadManifest

bool

マニフェストを再ロードするかを指定します。  
未指定の場合は true。

戻り値​
--------------------------

*   void

注釈​
-----------------------

実行中のエクステンションはリロードしません。  
また、DLL形式のエクステンションについては、リロードしてもコードは変わりません。
