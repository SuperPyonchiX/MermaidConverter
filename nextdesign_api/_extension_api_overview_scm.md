チーム開発
=====

説明​
-----------------------

チーム開発向けに構成管理システムとの連携操作を提供するAPI群です。

エリアに属するAPI​
-----------------------------------------------

名前

説明

[IScmRevision](_extension_api_NextDesign.Desktop_IScmRevision.md)

リビジョン情報へのアクセスオブジェクトです。

[IScmFileStatus](_extension_api_NextDesign.Core_IScmFileStatus.md)

物理ファイルの構成管理状態へのアクセスオブジェクトです。

[IScmManager](_extension_api_NextDesign.Desktop_IScmManager.md)

構成管理へのアクセスオブジェクトです。

[IScmRepositorySetting](_extension_api_NextDesign.Desktop_IScmRepositorySetting.md)

「リポジトリの管理」により登録したリポジトリ接続情報へのアクセスオブジェクトです。

[IEditPermissionResult](_extension_api_NextDesign.Desktop_IEditPermissionResult.md)

編集権限の取得/解放を要求した際の結果情報オブジェクトです。

[IScmChangePath](_extension_api_NextDesign.Desktop_IScmChangePath.md)

リビジョンでの変更対象要素情報オブジェクトです。
