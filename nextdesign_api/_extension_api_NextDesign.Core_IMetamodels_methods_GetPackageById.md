IMetamodels.GetPackageById メソッド
===============================

名前空間: NextDesign.Core

説明​
-----------------------

指定した識別子のパッケージを取得します。  
パッケージが見つからない場合は null を返します。

引数​
-----------------------

名前

型

説明

packageId

string

パッケージの識別子

戻り値​
--------------------------

*   [IPackage](_extension_api_NextDesign.Core_IPackage.md)
