IClass.GetFieldsByTag メソッド
==========================

名前空間: NextDesign.Core

説明​
-----------------------

指定されたタグが付与されたこのクラスのフィールドを取得します。  
タグ値が未指定の場合はタグの有無のみで評価します。

引数​
-----------------------

名前

型

説明

tag

string

タグ名

value

string

タグ値  
既定値は、null です。  
null または空文字が指定された場合は、フィールドに対するタグの有無のみで評価します。

戻り値​
--------------------------

*   IFieldCollection
