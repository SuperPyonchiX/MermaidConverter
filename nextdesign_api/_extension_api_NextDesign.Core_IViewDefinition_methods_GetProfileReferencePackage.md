IViewDefinition.GetProfileReferencePackage メソッド
===============================================

名前空間: NextDesign.Core

説明​
-----------------------

この要素を管理する基点パッケージを取得します。

基点パッケージは以下のいずれかのパッケージを指します。

*   プロファイルのルートパッケージ
*   プロファイル参照が記録されているプロファイル参照パッケージ
*   プロファイル参照で依存関係があるプロファイルの基点として記録されているパッケージ
