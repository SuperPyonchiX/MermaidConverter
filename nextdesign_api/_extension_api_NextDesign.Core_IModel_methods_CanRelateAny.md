IModel.CanRelateAny メソッド
========================

名前空間: NextDesign.Core

説明​
-----------------------

このインスタンスを指定したモデルと関連づけできるか調べます。  
このインスタンスのいずれかの参照フィールドで関連づけることができる場合はTrueを返します。  
なお、自身、もしくは関連づけするモデルが削除済みモデル、一時プロキシの場合はFalseを返します。  
なお、所有のフィールドに関しては対象としません。

引数​
-----------------------

名前

型

説明

opposite

IModel

関連づけするモデル  
null は指定できません。

戻り値​
--------------------------

*   bool

-----------------------

名前

例外クラス

説明
