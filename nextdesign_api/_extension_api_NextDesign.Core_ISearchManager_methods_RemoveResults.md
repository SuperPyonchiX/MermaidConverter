ISearchManager.RemoveResults メソッド
=================================

名前空間: NextDesign.Core

説明​
-----------------------

指定した検索結果をすべて削除します。  
既に削除済みの検索結果が含まれる場合は、その検索結果を無視します。

引数​
-----------------------

名前

型

説明

entries

IEnumerable<ISearchResultEntry>

検索結果の列挙

戻り値​
--------------------------

*   void
