IMetamodels.FindPackagesByTag メソッド
==================================

名前空間: NextDesign.Core

説明​
-----------------------

指定したタグが付与されたパッケージを検索します。

引数​
-----------------------

名前

型

説明

tag

string

タグ名

value

string

タグ値の文字列表現  
既定値は、null です。  
null または空文字が指定された場合は、パッケージに対するタグの有無のみで評価します。

戻り値​
--------------------------

*   IPackageCollection
