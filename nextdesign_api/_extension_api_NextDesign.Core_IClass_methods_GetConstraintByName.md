IClass.GetConstraintByName メソッド
===============================

名前空間: NextDesign.Core

説明​
-----------------------

このクラスで定義する指定された名前の制約を取得します。  
同じ名前の制約が複数ある場合は、最初に見つかった制約を返します。

引数​
-----------------------

名前

型

説明

name

string

制約名

戻り値​
--------------------------

*   [IConstraint](_extension_api_NextDesign.Core_IConstraint.md)
