イベント
====

説明​
-----------------------

購読可能なイベントとイベントの詳細情報にアクセスするAPI群です。イベントエリアごとに説明します。

イベントエリア​
--------------------------------------

名前

定義名

説明

[アプリケーション](_extension_api_overview_events_application.md)

application

アプリケーションの起動・終了を通知します。

[コマンド](_extension_api_overview_events_commands.md)

commands

コマンドの実行を通知します。

[プロジェクト](_extension_api_overview_events_project.md)

project

プロジェクトの作成・読み込み・保存・終了を通知します。

[モデル](_extension_api_overview_events_models.md)

models

モデルの作成・更新・移動・削除、および選択を通知します。

[エディタ](_extension_api_overview_events_editors.md)

editors

エディタの表示切替、選択変更を通知します。

[ページ](_extension_api_overview_events_pages.md)

pages

ページの変更（切り替え）を通知します。

[ナビゲータ](_extension_api_overview_events_navigators.md)

navigators

ナビゲータの表示切替、選択変更を通知します。

[情報ウィンドウ](_extension_api_overview_events_informations.md)

informations

情報ウィンドウ内のページ表示切替、表示要素選択変更を通知します。
