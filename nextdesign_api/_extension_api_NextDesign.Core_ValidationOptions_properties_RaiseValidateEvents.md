ValidationOptions.RaiseValidateEvents プロパティ
===========================================

名前空間: NextDesign.Core

説明​
-----------------------

モデル検証時イベントを発行するかどうかを取得、または設定します。  
既定値は true です。  
false を指定した場合、モデル検証時イベントが発行されません。

    bool RaiseValidateEvents { get; set; }

型​
--------------------

*   bool
